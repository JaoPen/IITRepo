PROJECT IDEA
  • Overview

    The project idea is home security using the Raspberry Pi. The prototype will be used in the grounds of private property which will serve as an additional tool to detect unlawful entry (burglary). Because of the high cost of some popular brands like NetGear Arlo and Swann, this prototype will serve its purpose at a fraction of the cost.


  • Motivation

    According to statistics, in Queensland, there are 31.5% families who are more concerned about home burglary. And in 2021, 73.5% would like to install a home security to reduce home insurance premium.

    Living in a quiet suburb north of Brisbane, who would have thought that the installation and cost of home security with the latest technology would be really expensive? Our house is equipped with home security, but it is wireless. We wanted to replace the existing ones and get it wired or as an additional with less cost. The problem is that it will be costly.

    My idea is to build home security using Raspberry Pi; it only costs roughly 5% of the total cost of what we have now. Though this project has already been completed by other experts, I think this is the possible solution for our home security.


    Statistics reference:

    https://www.budgetdirect.com.au/home-contents-insurance/research/home-burglary-statistics.html


  • Description

    The Raspberry Pi is a low cost, credit-card sized computer that plugs into a computer monitor or TV and uses a standard keyboard and mouse. It is a capable little device that enables people of all ages to explore computing, and to learn how to program in languages like Scratch and Python.

    This project idea is based on the need of having an additional home security system in order to have peace of mind, thinking that there is unlawful entry occurring and to reduce the premium of home insurance. There are similar projects that are undertaken by experts. I believe that constructing this prototype will be a big help to reduce part of our expenses. Some homeowners invest hundreds of dollars just to protect their property.

    This project comes with a PIR (Passive Infrared) sensor which will detect if there is movement, and a camera which will capture what the moving is. It will then be sent to a web server. In the web server, there are the list of how many log movements and photos taken since the last history was recorded. It has the capability to send information via email with the last history that occurred, together with an attached photo.

    Reference:

    https://www.raspberrypi.org/help/what-%20is-a-raspberry-pi/

    Skills required are a background in Python Programming, Electrical/Computer Engineering, and Raspberry Pi (preferably). An individual who has better understanding in hardware configuration, know how to source, select and negotiate for the best hardware packages with suppliers. Time management is also essential to complete the project.


  • Outcome

    If the outcome is successful, I will use the prototype as an additional element added to our home security. It will serve its purpose as an alarm and surveillance system without adding much cost to our expenses. Also, with additional home security, the probability of unlawful entry will decrease and become preventable, and our home insurance premium will reduce.

    This prototype will only be installed in our home but can be distributed (limited) to families and friends. It cannot be installed and distributed commercially.
